# Paketinhalt

Dieses Paket enthält:

- `meta-spec.md` – Meta-Spezifikation und Ausfüllvorlage für Schüler
- `fsd-prompt.md` – Prompt zur Erstellung eines englischen Functional Specification Document
- `implementation-plan-prompt.md` – Prompt zur Erstellung eines Implementierungsplans aus der FSD

## Empfohlener Ablauf

1. Schüler verfassen ihren Vorbereitungstext mit `meta-spec.md`.
2. Der fertige Text wird mit `fsd-prompt.md` in Codex eingegeben.
3. Codex erzeugt ein englisches Functional Specification Document in Markdown.
4. Dieses Markdown-Dokument wird gespeichert.
5. Danach wird das gespeicherte FSD mit `implementation-plan-prompt.md` verwendet, um einen Implementierungsplan zu erzeugen.
