# 2. Schülerfreundliche Schreibregeln

### Sprachliche Regeln
- Schreibe in **ganzen, klaren Sätzen**.
- Verwende **einfache, sachliche Sprache**.
- Definiere wichtige Begriffe.
- Vermeide unklare Wörter wie „modern“, „gut“, „praktisch“, wenn sie nicht genauer erklärt werden.

### Inhaltliche Regeln
- Schreibe lieber **konkret** als allgemein.
- Beschreibe **Beispiele**, wenn etwas sonst unklar bleibt.
- Nenne **Grenzen und Nicht-Ziele** ausdrücklich.
- Trenne **Problem**, **Ziel** und **Lösung** sauber.

### Denkregel
Frage dich immer:  
**„Könnte eine außenstehende Person mein Projekt auf Basis dieses Textes richtig verstehen?“**
