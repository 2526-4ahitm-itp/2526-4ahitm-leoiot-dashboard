Project Handover

The team took over an existing LeoGreen effort and refocused it on an operable school monitoring platform.

Problem Space

The project combines room-climate monitoring for classrooms with PV and energy-related telemetry.

Architecture

Sensors and generators publish to MQTT, Telegraf writes to InfluxDB, and dashboards consume the resulting data.

Sensor Hardware

ESP-based sensor boxes provide room measurements that can be visualized in dashboards and 3D views.

Tech Stack

The solution uses MQTT, InfluxDB, Grafana, Quarkus, Node services, and browser-based frontends.

Repository Flow

  • Docker Compose runs the platform locally.
  • Node services feed browser clients and PV ingestion.
  • GitHub Actions deploy the Astro site and rebuild the VM stack.
